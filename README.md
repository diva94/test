# Redesigning Poor Code

### Setup:
* clone the repository 
```
git clone 
```
* go inside directory
```
cd Redesigning Poor Code
```

* run main.py

``` 
python main.py
```
* On CL following options would appear:
1. Manage Books
2. Manage Users
3. Check Out Book
4. Check In Book
5. Exit

* select any one option to continue.
